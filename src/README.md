# src/

TypeScript source code for neorwc.

## Directory structure

| Directory | Description |
|-----------|-------------|
| `core/` | Configuration, TUI, file scanner, state management, templates, AI orchestration |
| `provider/` | AI provider implementations (OpenAI, Anthropic, Google, DeepSeek, Mistral, Cohere, Ollama) |
| `tips/` | Contextual tip system — loader, display, and static JSON tip files |
