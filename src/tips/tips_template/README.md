# src/tips/tips_template/

Static JSON tip files organized by category. These are loaded by `src/tips/tips.ts` and displayed contextually.

## Tip files

| File | Topic |
|------|-------|
| `api.json` | API design and integration tips |
| `bun.json` | Bun runtime tips |
| `cli.json` | CLI development tips |
| `css.json` | CSS and styling tips |
| `database.json` | Database design and query tips |
| `debugging.json` | Debugging techniques and tools |
| `design.json` | Software design patterns and principles |
| `docker.json` | Docker containerization tips |
| `git.json` | Git version control tips |
| `json.json` | JSON processing and manipulation |
| `linux.json` | Linux and shell tips |
| `markdown.json` | Markdown formatting tips |
| `neorwc_tips.json` | Neorwc-specific usage tips |
| `nodejs.json` | Node.js runtime and ecosystem tips |
| `npm.json` | npm package management tips |
| `performance.json` | Performance optimization tips |
| `python.json` | Python language and ecosystem tips |
| `security.json` | Security best practices |
| `testing.json` | Testing methodologies and tools |
| `typescript.json` | TypeScript language tips |
